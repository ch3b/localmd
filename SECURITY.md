# Security Policy

## Supported versions

localmd is pre-1.0 software. Security updates are provided for the latest version on the default branch.

## Reporting a vulnerability

Please do not open public issues for security vulnerabilities.

Instead:

1. Open a private security advisory on GitHub for this repository, or
2. Contact the maintainers through a private channel listed on the repo profile.

Please include:

- A clear description of the issue
- Reproduction steps
- Potential impact
- Any suggested mitigation

We will acknowledge reports as quickly as possible and work on a fix.
